// 01 — About Me

코드로 문제를
해결하는 개발자

데이터가 흩어져 있어서 의사결정이 늦어지는 상황에 관심이 많습니다. SIGNAL 프로젝트에서 글로벌 뉴스 크롤링 파이프라인을 직접 설계하고, BERT 기반 감성 분석 모델을 미세조정해 위험 신호를 자동 분류하는 시스템을 구현했습니다.

수집부터 분석, 서비스 연동까지 end-to-end로 직접 구현하는 경험을 통해 소프트웨어 전반을 이해하는 개발자를 목표로 합니다.

01

데이터 기반 사고

직관보다 수치로 판단합니다. Z-Score 이상탐지로 감정적 판단 없이 위험도를 정량화했습니다.

02

End-to-End 구현

데이터 수집, 전처리, 모델링, 서비스 연동까지 한 프로젝트 내에서 전 과정을 경험했습니다.

03

실행력

7주 안에 기획부터 발표까지 완주했습니다. 완벽보다 동작하는 결과물을 먼저 만듭니다.

크롤링 파이프라인 설계

11개 글로벌 매체에서 뉴스를 안정적으로 수집하는 자동화 배치를 APScheduler로 구현. 수집 실패 예외 처리와 중복 데이터 필터링 로직 직접 설계.

핵심 구현

// 01 — Project Overview

프로젝트 한눈에 보기

뉴스와 시장 데이터를 AI로 통합 분석해, 중소 무역기업이 위기를 직관적으로 파악하고 선제 대응하도록 돕는 플랫폼입니다.

01 — PROBLEM

무방비 중소 무역기업

수출입 중소기업의 88%는 별도의 환리스크 관리를 하지 않으며, 공급망 위기 발생 시 즉각 대응 가능한 기업은 단 33.5%에 불과합니다.

02 — SOLUTION

AI 기반 실시간 조기경보

BERT 임베딩, 실시간 환율·원자재 파이프라인, Z-Score 이상탐지를 결합해 위험 신호를 3단계로 등급화하고 즉시 알림을 전송합니다.

03 — IMPACT

데이터 기반 의사결정 지원

글로벌 리스크 히트맵으로 3초 안에 위험국 파악, 개인화 키워드 기반 뉴스 피드로 기업 비즈니스에 직결된 정보만 제공합니다.

04 — SCALE

구독형 수익 모델

비로그인·일반·프리미엄 3티어 구조로 설계. 프리미엄에서 무제한 키워드 설정, 위기 이메일 알림, 맞춤 대응 전략 제안 제공.

// 03 — Solution Architecture

어떻게 작동하나

4단계 자동화 파이프라인과 통합 리스크 스코어링으로 위험 신호를 정량화합니다.

01

데이터 수집

11개 글로벌 매체 뉴스와 7개 환율·원자재 지표를 1시간 단위로 자동 수집합니다.

02

AI 특징 추출

BERT + Nori 기반 키워드 추출 엔진으로 핵심 위험 시그널을 정밀 파악합니다.

03

위험도 등급화

Z-Score 기반 이상탐지와 상관 알고리즘으로 리스크를 주의·경고·심각 3단계 분류합니다.

04

스마트 딜리버리

사용자 맞춤 분석 결과를 실시간 이메일 Push 및 대시보드 리포트로 전송합니다.

// 05 — Risk Classification

3단계 위험도 분류

감정적 판단을 배제하고 데이터 기반 임계치로 리스크를 객관적으로 등급화합니다.

● 1단계 — 주의

모니터링 구간

Signal Score가 정상 범위를 소폭 이탈한 상태. 관련 뉴스 피드 및 지표 변동을 대시보드에 노출하여 능동적 모니터링을 지원합니다.

● 2단계 — 경고

대응 준비 구간

복수 지표에서 임계치 이탈 감지. 이메일 알림 발송 및 관련 국가·산업의 심층 리포트를 자동 생성하여 대응 시나리오를 제시합니다.

● 3단계 — 심각

즉각 대응 구간

공급망 단절·환율 급변 등 복합 위기 감지. 실시간 Push 알림과 함께 헤징 시나리오 및 대응 가이드를 즉시 전달합니다.

// 06 — SWOT Analysis

SWOT 분석

프로젝트의 강·약점과 시장 기회·위협을 분석했습니다.

Strengths
강점
  • BERT 기반 고성능 자연어 처리 엔진
  • 환율·원자재 실시간 자동 파이프라인
  • Z-Score 이상탐지로 객관적 임계치 산출
  • 중소기업 특화 초개인화 서비스
Weaknesses
약점
  • 글로벌 전 국가 데이터 커버리지 확장 필요
  • 유료 모델 초기 레퍼런스 확보 단계
  • 모델 학습 데이터 라벨링 공수 필요
Opportunities
기회
  • 공급망 불확실성 상시화로 수요 급증
  • SME 대상 디지털 전환 지원 정책 활성화
  • KOTRA 등 공공기관 수출 리스크 지원 수요
Threats
위협
  • 기존 대형 금융 데이터 벤더의 서비스 확장
  • 빅테크 기업의 AI 기반 시장 진입 가속화
  • 뉴스 API 비용 및 데이터 품질 의존성
